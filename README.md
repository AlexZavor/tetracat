# TetraCat
###### Tetra(4) - Cat, Way better than tennis

Tetris Clone designed for the GBA using Tonc and DevKitARM to create a homebrew game. Also built to help test my own <a href="https://github.com/AlexZavor/AC_GBA_Emulator">Emulator</a> and just have fun. I love game design and working close to the hardware.

```text
Art made with   - Aseprite
Written in      - C
Tested on       - 3DS**
SFX made with   - Bfxr
Music made with - N/A (what a problem :/)
```
** I have a GBA just no flashcart, but the 3DS uses hardware emulation (for GBA VC Games) so it should be hardware accurate.
